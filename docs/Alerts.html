poll()

Performs a JSONP "long poll" (a "reverse ajax" technique) This is the main code that interacts with the server to retrieve new information. With long polling, the XHR request is blocked on the server side (and kept open on the client) until there is new data to send. It sends the data, closes the connection, and initiates a new connection to wait for data again. JSONP is used to allow for cross-domain requests.
